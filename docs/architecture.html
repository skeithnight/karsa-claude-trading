Key observations:
• Trading Bot has no direct Bybit access — only Crypto Bot goes through WARP
• Both bots share the same Redis/Postgres but have separate Telegram tokens
• 9router is optional — falls back to direct Anthropic API if unavailable
• WARP healthcheck pings Bybit API to confirm geo-unblocked connectivity
• Orchestrator drives both equity scans AND crypto scans via cron
📦 7 containers 🔗 1 bridge network (trading-net) 💾 3 named volumes (redis, postgres, 9router) ⏰ 14+ APScheduler cron jobs 🛡️ 8 crypto risk gates 📊 3 market groups (IDX/US+ETF/Crypto)
